# Paint-App


## How to install
```
git clone
```
In the project directory:
```
npm install
```
To run:
```
npm start
```
## Features:

### Adding text boxes by dragging and dropping
- [x] Font size
- [x] Font style
- [ ] Text Color

### Add pictures from local storage
- [x] Can be moved to what position you want by dragging
- [ ] Picture resizeable

### Use a paintbrush for coloring
- [x] Brush color
- [x] Brush opacity
- [x] Brush width
